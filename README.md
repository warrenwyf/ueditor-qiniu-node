ueditor-qiniu-node
=================

支持UEditor连接到七牛的NodeJS服务



----------



开发环境
-------


#### Mac下安装

```bash
$ brew install node
$ cd <project-path>
$ sudo npm install
```


#### CentOS下安装

```bash
$ sudo yum install nodejs npm
$ cd <project-path>
$ sudo npm install
```


#### 启动服务

```bash
$ cd <project-path>
$ node main.js
```


#### 确认服务运行

在浏览器中访问[/info](http://localhost:3999/info)


#### 运行单元测试

```bash
$ cd <project-path>
$ nodeunit xxx_test.js
```
